# Generalizing deepfake video detection for disinformation mitigation: feature fusion and a novel specialized deepfake video benchmark dataset.

This repository contains the code for the Bachelor Thesis: **"Generalizing deepfake video detection for disinformation mitigation: feature fusion and a novel specialized deepfake video benchmark dataset"**

## Abstract 
With the surge of new video generation models, the problem of deepfake detection has become even more relevant than before. Easy access to novel generative AI systems capable of creating hyper-realistic fake content creates an imminent threat to society.

In this thesis, we formulate and empirically validate the hypothesis that recent state-of-the-art deepfake detectors fail to generalize to novel video generators. We introduce a controlled, video-only benchmark dataset, **MIDD (MisInformation Deepfake Detection)**, containing videos generated by six modern video generators in public-speaking setups. Evaluation on MIDD benchmark further confirms that existing academic protocols are outdated and do not support cross-generator generalization.

To address the limitations of current deepfake detectors, we propose using **feature fusion** of several modalities—specifically monocular depth estimation, curated point-tracking, and perception encoding—as a way of improving generalization. We systematically evaluate this approach, demonstrating that an additional visual backbone with a late feature-fusion strategy improves the baseline and increases overall detection performance, achieving state-of-the-art results on the MIDD benchmark.

## Repository Structure

The project is organized into three main components (each with detailed `README.md` instructions):

### 1. [Feature Fusion (`feature-fusion/`)](./feature-fusion/)
Core codebase for training and evaluating multi-feature fusion deepfake detection models.
- **Features Extracted:** DFD-FCG, Depth Anything V2, Perception Encoder (PE), CoTracker 3.
- **Fusion Mechanisms:** Concatenation, Cross-Attention, Gated, Late Fusion.
- **Usage:** Handles training (`train_exp.py`), evaluation (`evaluate.py`), inference (`infer.py`), and visualization scripts.

Visit folder's README for more details


### 2. [Video Generation (`video-generation/`)](./video-generation/)
Scripts to generate the synthetic subset of the MIDD dataset using modern diffusion-based generation models.
- **Models:** Wan 2.2 and HunyuanVideo 1.5.
- **Usage:** Automated environment setup, Text-to-Video (T2V), and Image-to-Video (I2V) capabilities.

Visit folder's README for more details


### 3. [Video Processing Pipeline (`video-processing-pipeline/`)](./video-processing-pipeline/)
The comprehensive data pipeline for curating, processing, classifying, and filtering the raw videos.
- **Features:** Face/text detection, duration curation, semantic classification, and multimodal prompt generation using Gemini models (for conditioning generators).

Visit folder's README for more details
